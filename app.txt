APK Generation:
EAS Build (recommended, cloud-based)
Expo provides a free cloud build service called EAS Build. You would:

Create a free account at expo.dev
Install the EAS CLI on your own machine (npm install -g eas-cli)
Run eas build -p android --profile production from the artifacts/mobile folder
Download the resulting .apk file when the build finishes (usually 10–15 minutes)
This produces a proper signed APK you can sideload directly onto your Android device.
